# S7 Home Manager
It is declared configuration for OS environments.

> Inspired by runbook [simple-homemanager](https://github.com/Evertras/simple-homemanager)

## Getting started
1. Clone this repository

2. Init environment
```sh
make init TARGET_HOST=your_target_host
```

3. Apply configuration changes
```sh
make
```
